# 欢迎开始你的博客之旅

## 引言

欢迎来到这个现代化的博客系统！这里是一个基于 Next.js 14 和 TypeScript 构建的全栈博客应用，提供了丰富的功能和优秀的用户体验。

## 主要特性

### 🚀 快速启动
- 基于 Next.js 14 的现代化架构
- TypeScript 提供类型安全
- Tailwind CSS 实现美观的界面设计

### 📝 内容管理
- Markdown 支持
- 实时预览
- 代码高亮
- 图片上传

### 🎨 用户界面
- VS Code 风格设计
- 暗色主题优先
- 响应式布局
- 平滑动画效果

## 快速开始

1. **安装依赖**
   ```bash
   pnpm install
   ```

2. **启动开发服务器**
   ```bash
   pnpm dev
   ```

3. **访问应用**
   打开浏览器访问 `http://localhost:3005`

## 技术栈

- **前端框架**: Next.js 14 + React 18
- **编程语言**: TypeScript
- **样式框架**: Tailwind CSS
- **状态管理**: React Hooks
- **文件系统**: Node.js fs API

## 目录结构

```
├── app/              # Next.js App Router
├── components/       # React 组件
├── lib/              # 工具函数和配置
├── public/           # 静态资源
└── styles/           # 全局样式
```
